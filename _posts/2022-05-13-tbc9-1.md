## 9. 문자 입출력과 입력 유효성 검증
### 9.1 함수가 필요할 때
#### 9.1.1 명함 출력 

1. 이와 같은 명함을 만드려면 단순하게는 아래와 같이 하면 된다. 

```
********************
    Gil-Dong Hong
    Seoul, Korea
********************

C:\Users\...\Practice\x64\Debug\Practice.exe (process 25124) exited with code 0.
Press any key to close this window . . .
```

```c
#include <stdio.h>
#include <stdlib.h>

int main() {

	printf("********************\n");
	printf("    Gil-Dong Hong\n");
	printf("    Seoul, Korea\n");
	printf("********************\n");

	return 0;
}
```
